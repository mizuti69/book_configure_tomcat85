# Apacheとの連携設定

#### AJP設定
tomcatとの連携でフロントエンドにapacheを採用し、AJPコネクターを利用する場合、  
apacheの`mod_proxy_ajp`を利用します。  

**連携設定**  

URL単位で連携方法を定義します  

```
  <Location />
    ProxyPass ajp://localhost:8020/
  </Location>
```

**連携除外設定**  
下記例では外部からのリバースプロキシを無効化し、  
URL「/hogehoge/」はtomcatに連携しないよう除外しています。  

```
  # Proxy applicaiton path
  ProxyRequests Off
  ProxyPass /hogehoge/ !
```
